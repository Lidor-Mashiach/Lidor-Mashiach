## 🔬 Research

### Thesis — Long-Term Memory in HTM Models *(ongoing, private repository)*

Investigating multi-timescale memory architectures in HTM — enabling models to retain learned patterns over extended time periods without forgetting. The work extends a pyramid architecture (PyHTM) validated across 6 real-world datasets spanning industrial control, spacecraft telemetry, server monitoring, and network security.

Core question: what architectural connections between short-term and long-term memory horizons produce the best results across diverse temporal domains?

Comparison models include USAD, DAGMM, LSTM-VAE, OmniAnomaly, MSCRED, THOC, and GDN.

## 🚀 Featured Projects

### 🔍 Focal Search: Searching for Optimum
> Research conducted as part of the M.Sc. program at BGU.  
> Introduced **Ratio-Tuned Focal Search (RFS)** — a parameterized framework replacing fixed expansion strategies in bounded-suboptimal search with a tunable parameter φ ∈ [0, 1].  
> Evaluated across 5 domains. Pure Focal Search (φ=0) achieves up to **35,360x** reduction in node expansions over A*.  
> A robust non-linear relationship (polynomial degree 3, **R²=0.9333**) was identified between domain difficulty and focal search benefit — suggesting search efficiency is a predictable function of state-space complexity, not a random empirical observation.

🔗 [View Repository](https://github.com/Lidor-Mashiach/Focal_Search)

### 🌧️ Short-Term Rainfall Prediction Across the Tropical Belt
> Research conducted as part of the M.Sc. program at BGU.  
> Data-driven framework for hourly rainfall occurrence prediction across the entire tropical belt using ERA5 reanalysis data.  
> Evaluated Logistic Regression, GAM, and XGBoost across four forecast horizons (1h, 3h, 12h, 24h).  
> XGBoost achieves ROC-AUC of **0.976** at 1h and maintains strong performance at 3h — well beyond simple nowcasting. Spatiotemporal features dominate predictive importance across all models and horizons.

🔗 [View Repository](https://github.com/Lidor-Mashiach/Tropical-Predictor)
